# 💣 Bomb Defuse Game

An immersive browser-based bomb defusal puzzle game built using HTML, CSS, and JavaScript.

Players must solve radio-frequency puzzles, discover hidden passcodes, and defuse the bomb before the timer reaches zero.

---

## 🎮 Features

* ⏳ Real-time countdown timer
* 📻 Interactive radio frequency puzzle
* 🔐 Keypad password system
* 📖 Animated notebook with hints
* 🔊 Horror-style sound effects
* 💥 Explosion sequence with screen shake
* 👻 Hidden jumpscare easter egg
* 🎧 Dynamic audio system
* 🧩 Multiple hidden secret codes

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎯 Gameplay

### Objective

Defuse the bomb before time runs out.

### How to Play

* Turn on the radio
* Tune frequencies
* Listen for clues
* Enter passcodes using the keypad
* Press `Y` to submit
* Avoid wrong passwords or lose time
